# SimpleKit Investment Fee Calculator

A static SimpleKit app that compares low-fee and high-fee investing over time so users can see how MER drag affects long-term wealth.

## What it includes

- SimpleKit-aligned hero, cards, buttons, spacing, footer, and floating support action
- Live portfolio comparison with Portfolio A vs Portfolio B
- Advanced assumptions for inflation, tax drag, contribution growth, lump sum timing, and compounding frequency
- Responsive SVG charts with no external chart dependency
- Milestone insights, scenario testing, Canadian context, related tools, and SEO FAQ content
- Existing Google Analytics head snippet preserved from the template
- Static metadata files for deployable hosting

## Local use

1. Open [index.html](/Users/AshleySkinner/Documents/00_Engineering/04_Code/37_Investment Fee SimpleKit/index.html) directly in a browser, or serve the folder with a small static server.
2. Adjust inputs in the calculator.
3. Use `Load sample data` to show a realistic long-term comparison.

## Notes

- State is saved locally in the browser under `simplekit.investmentFeeCalculator.v1`.
- The model is intentionally simplified and should be treated as educational planning support, not financial advice.
- `robots.txt`, `sitemap.xml`, and social URLs currently assume the intended production path is `https://simplekit.app/investment-fee-calculator/`. Update those if deployment uses a different URL.
